# Birthday Website

A premium single-page birthday experience for Maryam.

## Files

- `index.html` — Landing page and scene structure.
- `style.css` — Glassmorphism, gradients, and animation styles.
- `script.js` — Scene flow, typed text, audio control, and confetti.
- `assets/music.mp3` — Background audio file.
- `assets/cake.png` — Cake illustration asset.
- `assets/stars.svg` — Decorative star overlay.
- `assets/heart.svg` — Heart motif asset.

## Usage

Open `index.html` in a modern browser. Tap **Open Your Surprise** to begin, then enjoy the animated birthday sequence.

## Notes

The design uses no JavaScript frameworks and only Google Fonts for typography.
